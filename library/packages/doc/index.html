Packages Manipulation

This is a set of basic routines for manipulating packages.

Authors: Martin Vidner <mvidner@suse.cz>, Michal Svec <msvec@suse.cz>.

Modules

The module Package is the default one, it should work on the target system (it correctly differentiates between normal and autoinstallaton mode).

PackageSystem is for situations when you need immediate action (for example to start the client).

  • PackageSystem
